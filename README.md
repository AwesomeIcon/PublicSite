# PublicSite

## 准备工作

* Sublime text

* git

* mysql

### 环境安装

* 安装django1.4.21环境

```bash
pip install django==1.4.21
```

* [安装python2.7](https://www.python.org/downloads/release/python-2711/)

* python-mysql 库

```bash
pip install python-mysql
```

## 介绍

* 使用django框架进行开发

* 基于python 2.7
